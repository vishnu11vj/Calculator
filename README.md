# 🧮 Calculator
A sleek, fully functional calculator with a dark purple theme built using pure HTML, CSS, and JavaScript.
## Live Demo
Open `index.html` in any browser — no installation needed.
## Features
- Basic arithmetic operations (add, subtract, multiply, divide)
- Percentage and sign toggle (+/-)
- Expression display showing current operation
- Glowing dark UI with smooth button interactions
## Built With
- HTML5
- CSS3
- Vanilla JavaScript
